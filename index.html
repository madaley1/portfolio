About Me

Hey, I’m Morgan. I’m a Software Engineer with Full-Stack experience, as well as a Web Application Developer. I also have quite a lot of experience with Graphic Design. I got my degree from ASU's Ira. A Fulton Schools of Engineering but the majority of my experience has been extracurricular & professional in nature. I enjoy playing around and seeing the boundaries of what a website can do, as well as expanding my knowledge of languages. I'm currently employed with ASU's College of Liberal Arts and Sciences as a Website Technician, working on their internal Drupal distribution. I spent my childhood internationally, living in various European and Asian countries, and came back to America for my High School and Higher Education. Through a lot of this period I learned quite a few creative hobbies, including video production, graphic design, digital art, live streaming, and others, but the only one that has really stuck around was development. Overall my goal is to continue learning and growing to the limit of what I am capable of.

Past Projects

TheMovieDB API Project

This project was my introduction to using RESTful API with JSON Formatted Data. It is a Single-Page Application that returns four recently released movies, as well as giving you an option to see some movies that were released on your birth year (or any year in general, but was intended for birth year).

Stock Comparison API

This stock comparison Single Page Application was built using the IEXCloud API for Stock Data, using the simulated environment. It also uses the google charts API for rendering the datas bar chart. It allows the user to add as many current stock prices as requested, and updates every 5 seconds.

The KenpoNet

The KenpoNet is a website I found when researching the Martial Art I currently hold a black belt in. I decided to redesign & relaunch it after being unable to make contact with the owner via all available methods, and getting the contact information from the web host directly. Myself and a couple friends are creating a drupal site to host this information, and will be launching it upon completion and another round of attempts to contact the original owner, in case they have intent to continue work on their site.
